Sale Discount Limit

Aktiv Software has prepared the Sale Discount Limit module with the aim of restricting over discount based on Sales Group like Sales Manager can giver maximum 15% Discount, Sales User can give maximum 10% Discount.

Configurations

  • Go to Sales > Configuration > Sales Discount Limit
  • Select Sales User Group and Put Discount in Percentage

Restrict Over Discount

Create a user and give an access right of the Sales Manager and Discount on lines
